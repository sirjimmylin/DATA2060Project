# DATA 2060 Project: Gaussian Naive Bayes
## Project Overview

This machine learning project is a group project for the class DATA 2060: Machine Learning: From Theory to Algorithms. In this project, our group recreated from scratch a custom implementation of the Gaussian Naive Bayes (GNB) algorithm. 

[Scikit-Learn's Implementation For Reference](https://github.com/scikit-learn/scikit-learn/blob/1eb422d6c5/sklearn/naive_bayes.py#L151)
